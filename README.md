## Usage

```js
$ npm install por-extenso
por-extenso@1.1.1 node_modules/por-extenso
$ node
> var porExtenso = require('por-extenso')
undefined
> porExtenso('9')
'nove'
> porExtenso(123)
'cento e vinte e três'
```
